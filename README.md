# Padel Score (2025)

Want to track your score during padel match? Use this app!

Live demo [here](https://padel-score.itpulse.pl).

The app was made with Next.js 16 in 2025.

Copyright 2025 IT Pulse Mikołaj Pęcak. All rights reserved.

## Getting started

To run the project:
1. `npm ci` to install dependencies
2. `npm run dev` to run the app locally (http://localhost:3000 by default)
